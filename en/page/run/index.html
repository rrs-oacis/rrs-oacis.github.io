Upload Agent

Upload Agent data used for simulation to RRS-OACIS.

Move to operation screen

Click "Agents" from the sidebar.

(The following images can be magnified)

Or, to the browser

http://localhost:3080/agents

to access

Operation

step 1

Select the Agent to upload

Step 2

Enter the name of the agent

step 3

Press the button to upload Agent data to RRS-OACIS.

step 4

Upload completed when displayed as above

Tips
The name of Agent can be any string

Activate functions for simulation

Activate disabled simulation features by default

Move to operation screen

Click Settings from the sidebar.

Or to the browser

http://localhost:3080/settings

to access

Operation

step 1

Click "Apps" to move to move to the function list screen

step 2

Select Run

step 3

Activate the function by pressing the "Enabled" button

step 4

Once activated, activation is complete.

Perform simulation

I try to simulate with uploaded data

Move to operation screen

Click "Run" from the sidebar.

Or, to the browser

http://localhost:3080/run

to access

Operation

step 1

Select Map from list

step 2

Select Agent from list

step 3

Press the button to start the simulation.

step 4

Simulation was registered when displayed as above.

step 5

Simulation is finished when "Finished"appears in status as above.

Simulation is running when status is displayed as "running". When "none", the simulation has not started yet.

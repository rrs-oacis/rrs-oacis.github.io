To enable the extension.

Video

RRS-OACIS in various feature is disabled by default. It is possible to use by activating the setting screen.

Move to the operation screen

In the browser

http://localhost:3080/settings

Go to.

Operation

step 1

Move by clicking on the "Apps" in order to move to the function list screen

step 2

This time, select the Run

step 3

Enable the feature by pressing the "Enabled" button.

step 4

When you see as above is the activation is complete.

インストール方法

RRS-OACISを利用するには

  1. RRS-OACISのセットアップ
  2. シミュレーション用クラスターのセットアップ

の両方を行う必要があります.


推奨環境として

パソコンのスペックとしては

  • CPUは第4世代以降のIntel Core i7
  • メモリーは16G
  • 有線のギガビットイーサー

が必要です.

パソコンの台数は

  • RRS-OACISをインストールするバソコンが1台　(A)
  • クラスター化するパソコンが4の倍数台　(B)

が必要になります.


最低環境として

パソコンのスペックとしては

  • CPUは第3世代以降のIntel Core i7(3.0GHz)
  • メモリーは8G
  • 有線のファストイーサー

が必要です.

パソコンの台数は

  • パソコンが1台　(A) (B)

で動作が可能です.

Images

RRS-OACISとクラスターの物理的な配置はこのようになっています．(A)がRRS-OAICSをセットアップするサーバーで(B)がクラスターになるサーバーです． それぞれのパソコンは常に接続されている必要があります．


(2)シミュレーション用クラスターのセットアップはブラウザ上でおこないます.

RRS-OACISのインストールが完了してから作業をしてください．

Tips
同じネットワーク内なら3080番ポートで別のパソコンから操作することができます.

(1)RRS-OACISのインストール

Dockerというツールを使ってRRS-OACISがインストールされた仮想環境を手軽に導入することができます. Linuxだけでなく、Windows、macOSにも導入することができます. 手順の概要は以下の通りです.

OSはUbuntuを想定して説明します．

必須環境

  • Docker (checked 17.09.0-ce)
  • Dockerが動作するOS
  • インターネット接続

Dockerのインストール (※ インストール済みの方はスキップしてください)

DockerをUbuntuにインストールします. 下記のリンクを参考にインストールしてください.

UbuntuにDockerをインストール

※ Windowの方はこちらから、 Macの方はこちらからインストールしてください.

RRS-OACISのインストール

以下のスクリプトを実行することでRRS-OACISが導入された環境を構築できます．

※構築には数分かかる場合があります．

※DockerがインストールされたPCで実行してください．

sudo docker run --name rrsoacis -p 3080:3080 -p 3000:3000 -p 49138:49138/udp -dt rrsoacis/rrsoacis

下のように表示されれば導入完了です． 

Digest: sha256:(※UID)
Status: Downloaded newer image for rrsoacis/rrsoacis:latest

ブラウザで 

 http://localhost:3080/

にアクセスして下のように表示されるか確認します．

（以下の画像は拡大出来ます）

Tips
RRS-OACISをインストールするOSに制限はありません．Dockerが動作するならどのOSでも動作できます.

(2) クラスターのセットアップ

OSはUbuntuを想定して説明します．(macOSでも同様の操作で実行できることを確認しました．)

必須環境

  • Ubuntu 16.04.3 LTS
  • Oracle JDK (checked 1.8.0_40)
  • OpenSSH
  • パスワードの設定されたUbuntuのローカルアカウント

設定画面に移動

ブラウザで

    http://localhost:3080/settings-clusters

にアクセスします.

クラスターの設定

ステップ1

クラスターにするPCのユーザー名とIPアドレスを入力します．形式は「ユーザー名@IPアドレス」になります．

ステップ2

シミュレーションのログファイルの保存形式を設定します．クラスターのOSが対応している形式を選択します． 使用可能なのはgzipと7zipとzipがあります.

※Ubuntuの場合はデフォルトのgzip形式で問題ありません．

ステップ3

クラスターのパスワードを入力します．

※Ubuntuのアカウントのパスワードです．4台とも同じパスワードを想定しています．

ステップ4

Addボタンを押して追加します．

ステップ5

上のように表示されたら設定完了です．

Tips
クラスターは1~3台だけで構築する事も可能です．4台用意する必要はありません．

Agentのチュートリアル

基本的な使い方は とりあえず実行 を参考にしてください.

Agentをダウンロードする

RRS-OACISにアップロードしたAgentのダウンロード方法を説明します.

操作画面に移動

ブラウザで 

http://localhost:3080/agent/(agentのユニークID)

にアクセスします.

Agentの画面からLinkで移動できます.

操作

ステップ1

「Download Zip」のボタンを押します.

ステップ2

Agentのデータがダウンロードできます.

Agentをアーカイブする

RRS-OACISにアップロードしたAgentをアーカイブする方法を説明します.

アーカイブされたAgentはシミュレーションの選択画面などで表示されなくなります.

操作画面に移動

ブラウザで 

http://localhost:3080/agent/(agentのユニークID)

にアクセスします.

Agentの画面からLinkで移動できます.

操作

ステップ1

「Archive」のボタンを押します.

ステップ2

上の画像のようにArchivedの項目がTrueになっていればアーカイブ成功です.

アーカイブしたAgentを表示する

アーカイブしたAgentの表示方法を説明します.

操作画面に移動

ブラウザで 

http://localhost:3080/agents_archived

にアクセスします.

上の画像のようにアーカイブされたAgentのデータがリストで表示されます.
